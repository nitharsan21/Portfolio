AppliRv - Application JavaFx

Contexte

L’entreprise GSB est issue de la fusion de l’entreprise pharmaceutique américaine Galaxy et du conglomérat européen Swiss Bourdin.

Elle organise des visites dans les laboratoires afin de promouvoir leurs recherche pharmaceutique. Les visiteurs doivent rendre compte de leur visite après chacune d'elle..

Dans le but de faciliter le contrôle des délégués régionaux et d'avoir une vision plus claire de l'activité présentée, l'entreprise voudrait mettre en place une application JavaFx (client lourd pour ordinateur) permettant de v isualiser les rapports de visite et les praticiens répertoriés.

Environnement technologique

Généralités :
  • L'application a été développé à partir d'une machine virtuelle Debian7.
  • Le client lourd dédié à la vérification des rapports de visite sera développée à l'aide d' Eclipse.
Langages utilisés :
  • Java.
  • SQL (MySQL).
Outils :
  • Architecture composé de :
    • Un SGBD MySQL5
    • Un environnement Java JDK8
    • Un outil de connexion au SGBD driver JDBC (mysql connector)
  • logiciel de virtualisation VMware Workstation 12
  • IDE Eclipse

Fonctionnalités de l’application

  • Authentification du délégué régional
    • Navigation
      • Consulter un rapport de visite
      • Consulter la liste des praticiens
      • Déconnexion
