AppRv - Application Android

Contexte

L’entreprise GSB est issue de la fusion de l’entreprise pharmaceutique américaine Galaxy et du conglomérat européen Swiss Bourdin.

Elle organise des visites dans les laboratoires afin de promouvoir leurs recherche pharmaceutique. Les visiteurs doivent rendre compte de leur visite après chacune d'elle.

Dans le but d'améliorer la communication auprès des praticiens et d'avoir une vision plus claire de l'activité présentée, l'entreprise voudrait mettre en place une application Androïd permettant de saisir et centraliser les rapports de visite.

Environnement technologique

Généralités :
  • L'application a été développé dans un environnement Linux.
  • L'application mobile dédiée à l'enregistrement des rapports visites sera développée grâce à l'environnement Java JDK8 et au SDK d'Android.
Langages utilisés :
  • JAVA.
  • XML.
  • Python.
  • SQL (MySQL).
Outils :
  • Architecture REST (Python/Flask) composé de :
    • Un serveur Linux (Debian7)
    • Un client Linux
    • Un client Android
  • Virtualisation du serveur : VMware Workstation
  • Environnement Java : jdk8
  • Bibliothèques : JSON/GSON
  • SDK Androïd
  • Vérification des routes : Poster (Plugin/Addon Firefox)
  • IDE : Android Studio
